HTML.

I provide custom HTML coding for all your HTML needs. H1's? You got it. Paragraph elements? You got it. Div tags? You bet! I can even add styling to your content right in your HTML (though I don't recommend it). I can also review your HTML for ARIA screen reader optimization, as well as update your HTML for both accessibility and search engine considerations.

CSS.

Inline styling not your thing? I've got you covered! I can custom style your HTML content using CSS classes, ID's, or HTML tags from your HTML content. I can even combine classes and ID's to create styling for multiple HTML elements at the same time or to target very specific parts of your HTML. Looking for fancy buttons, or simple animation or hover effects? CSS is the way to go!

JavaScript.

If you want to make your site do awesome modern things, like allow your users to interact with your site in lots of interesting ways, you need JavaScript. I can use JavaScript and jQuery to dynamically populate your page with customized elements of all kinds using minimal HTML. With loops, functions, arrays, objects, and all of the power of JavaScript behind your site, the sky is the limit!

Bootstrap.

If you need a website quickly, with fun, interactive components and device-responsive grid layouts, you came to the right place. I can place professional looking navigation bars, buttons, carousels, user input elements, media objects and jumbotrons on your website in no time. If you're looking for something really fancy, ask me about the Bootstrap Scrollspy Component.
